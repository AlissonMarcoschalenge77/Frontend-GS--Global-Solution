# SmartAssist - Assistente Virtual Inteligente

## Descrição do Projeto

O **SmartAssist** é um assistente virtual desenvolvido para auxiliar estudantes e profissionais da área de tecnologia. O projeto integra inteligência artificial conversacional, design responsivo e interatividade, oferecendo uma experiência completa de aprendizado e orientação em programação.

Este projeto foi desenvolvido como parte da disciplina de **Front-End Design Engineering** da FIAP, aplicando boas práticas de HTML semântico, CSS modular, JavaScript interativo e versionamento com Git.

---

## Objetivo e Problema Resolvido

### Problema
Muitos estudantes e iniciantes em programação enfrentam dificuldades para:
- Obter respostas rápidas para dúvidas técnicas
- Encontrar orientação sobre carreira e próximos passos no aprendizado
- Acessar recursos confiáveis e atualizados sobre tecnologias
- Ter suporte disponível fora do horário comercial ou acadêmico

### Solução
O SmartAssist oferece um assistente virtual disponível 24/7, capaz de responder perguntas sobre programação, orientar sobre carreira em tecnologia e fornecer recursos de aprendizado curados.

### Impacto
Contribui com o **ODS 4 — Educação de Qualidade** ao democratizar o acesso ao conhecimento tecnológico.

---

## Tecnologias Utilizadas

- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Design responsivo com suporte a temas claro e escuro
- **JavaScript (ES6+)** - Interatividade, validação de formulários e controle de componentes
- **IBM Watson Assistant** - Inteligência artificial conversacional
- **GitHub** - Versionamento e colaboração

## Funcionalidades

### Responsividade
- Adaptação para Mobile (até 480px)
- Adaptação para Tablet (≥768px)
- Adaptação para Desktop (≥992px)
- Sem scroll horizontal ou quebras de layout

### Interatividade JavaScript
- **Menu Hambúrguer** - Funcional com controle de estado e reset no resize
- **Modal de Contato** - Abre/fecha com animações e trap de foco
- **Validação de Formulário** - Campos obrigatórios e validação de e-mail
- **Tema Claro/Escuro** - Alternância com persistência no localStorage
- **Acessibilidade** - ARIA labels, foco visível e navegação por teclado

## Como Executar o Projeto

1. Clone o repositório:
\`\`\`bash
git clone https://github.com/AlissonMarcoschalenge77/Frontend-GS--Global-Solution.git
\`\`\`

2. Navegue até a pasta do projeto:
\`\`\`bash
cd smartassist
\`\`\`

3. Abra o arquivo `index.html` no navegador ou use um servidor local:
\`\`\`bash
# Com Python 3
python -m http.server 8000

# Com Node.js (http-server)
npx http-server
\`\`\`

4. Acesse no navegador: `http://localhost:8000`
